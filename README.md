# Actividad 1 - Programación Web

Este repositorio contiene los ejercicios de la unidad 2 de la materia
de Programación Web haciendo uso de HTML, CSS, JavaScript y Bootstrap.

## Contenido

Cada archivo `ejercicioN.html` es un ejercicio. Los estilos
están en la carpeta `css/`, los scripts en `js/`, y las imágenes en `img/`.

## Para ver los ejercicios

Se puede navegar desde el [index.html](./index.html), que funciona como menú
con enlaces a cada ejercicio, o entrar directamente al archivo del ejercicio 
que se quiera revisar.
